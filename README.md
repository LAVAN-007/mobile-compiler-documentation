# 📱 Cross-Platform Mobile Compiler & Learning Dashboard

A robust, full-stack mobile-responsive web application designed to evaluate, compile, and run multiple programming languages while offering a structured learning roadmap for student developers.

---

## 🎯 Project Overview
This platform serves as an interactive coding ecosystem where users can write code, manage data structures, track their execution logs, and level up their problem-solving skills through curated milestones. 

> 🔒 **Repository Privacy Note:** The source code of this project is kept **Private** as it is currently being utilized in a live training environment with active student modules and real-time development updates. Technical walkthroughs or architectural code evaluation can be requested via email.

---

## 🚀 Key Features Demonstrated in Video

* **Multi-Language Execution Engine:** Integrated dynamic compiler supporting **Java, Python, and C** inputs with seamless live execution.
* **Structured Learning Paths:** Pre-configured algorithmic tracks divided into **Easy** and **Medium** modules (Basic I/O, Loops, Conditionals) to track student progress.
* **Advanced User Dashboard:** Personalized profile verification system with dynamic cloud database states to retrieve saved projects.
* **Modern UI/UX Architecture:** Mobile-first layout featuring custom dynamic keyboard support for coding symbols and interactive **Dark / Light Mode** switching.

---

## 💻 Tech Stack Used

* **Frontend:** React.js, Tailwind CSS, Responsive Mobile UI Layout
* **Backend Logic:** Java, Spring Boot Architecture, RESTful Web Services
* **Database & Hosting:** MongoDB Atlas (Cloud NoSQL), Hosted on Netlify & Render

## ⚙️ Architecture Flow
1. **User Authentication:** Simple email-based session login directing students to their tracking portal and personalized workspace.
2. **Challenge Selection:** Fetching problem datasets and saved workspaces from the cloud database cluster based on user parameters.
3. **Execution Pipeline:** Real-time asynchronous synchronization between the React code editor interface and Spring Boot backend execution consoles.
